REAL ESTATE PRICE PREDICTOR

https://real-estate-price-predictor-q775.onrender.com

This application provides instant real estate valuations based on key features like town, state, property type, parking space, bedroom and bathroom number. By moving beyond a simple notebook, this tool demonstrates how a machine learning model can be integrated into a functional, user-friendly interface.

How it Works:

Input: Users enter property details via the web form.
Processing: The backend cleans the input data and passes it to a pre-trained Random Forest Regressor.
Output: The model outputs a predicted price based on patterns learned from "Nigeria Housing and Pricing Dataset" by abdullahi yunus.

Tech Stack:

Machine Learning: Python, Scikit-learn (Random Forest), Pandas, NumPy.
Backend: Django
Frontend: HTML & CSS
Deployment: Render.

Key Features:

High Accuracy: Uses an ensemble of decision trees to handle non-linear relationships in housing data.
Responsive Design: Fully accessible via mobile or desktop browsers.
Scalable Deployment: Hosted on Render for reliable uptime.
